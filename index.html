1. Contenedor (wrapper)

El contenedor es un elemento envolvente que engloba todo el contenido de una página web. Su función principal es centralizar y controlar el diseño, creando límites o márgenes en la disposición visual. En CSS, suele utilizarse para establecer un ancho máximo y para agrupar otros elementos dentro de un bloque manejable.

2. Cabecera (header)

El header es la parte superior de una página web, y normalmente incluye el logotipo, el nombre del sitio, el título o el eslogan, así como elementos de navegación primarios como un menú. Es común que esta sección esté visible en todas las páginas de un sitio web.

3. Contenido (content)

El contenido es la parte principal de una página web, donde se presenta la información más relevante para el usuario, como textos, imágenes, videos, o cualquier otro tipo de datos. En esta sección se muestra el propósito principal de la página, como artículos, productos o servicios.

4. Menú (menu)

El menú o barra de navegación facilita la exploración del sitio. Suele contener enlaces a las diferentes secciones o páginas, proporcionando una estructura clara y accesible para los usuarios. Puede colocarse en la cabecera, en un lateral o incluso en un pie de página.

5. Pie (footer)

El footer es la parte inferior de una página web. Generalmente contiene enlaces adicionales, como políticas de privacidad, términos de uso, información de contacto, o enlaces a redes sociales. También puede incluir datos sobre la autoría o derechos de autor del sitio web.

6. Lateral (sidebar)

La barra lateral es una sección que suele colocarse a uno o ambos lados del contenido principal. Se utiliza para mostrar información secundaria o complementaria, como menús adicionales, widgets, anuncios, enlaces a artículos populares o formularios de suscripción.

7. Etiqueta div

El elemento div en HTML es un contenedor genérico utilizado para agrupar bloques de contenido o elementos. No tiene un estilo ni función propia, pero es clave para la estructura del diseño y el estilo de la página a través de CSS. Es muy versátil y permite separar secciones de una página para facilitar su disposición y estilización.
